\# 👨‍💼 Employee Performance Predictor using Machine Learning



\## 📌 Overview

This project is an end-to-end Machine Learning system that predicts employee performance (High / Medium / Low) based on various HR-related features.



It helps organizations make data-driven decisions for:

\- Promotions

\- Training programs

\- Employee retention

\- Performance evaluation



\---



\## 🎯 Problem Statement

Companies often struggle to evaluate employee performance objectively. This project solves that by building a predictive model using employee data.



\---



\## 💡 Solution

A complete ML pipeline that:

\- Generates synthetic employee data

\- Performs data preprocessing and analysis

\- Trains a classification model

\- Predicts employee performance

\- Displays results using a Streamlit web application



\---
